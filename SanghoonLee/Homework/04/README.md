# 숙제

```make
1. 오늘 학습한 내용을 복습하기
2. 이론적으로 복습하기
3. 그림을 직접 그려봅니다(그림 그린거 같이 제출하세요)
4. if문에 대해 기계어를 분석합니다.
5. for문에 대해 기계어를 분석합니다.
6. while문에 대해 기계어를 분석합니다.
   제어문에서는 rcx 레지스터가 활용되며 cmp(비교) 명령어와
   조건부 점프 jmp(jnz, jne, jle)등이 나타나게 될 것임
   코드가 단순한 것으로 하여 작업하세요(로직에 집중해야함)
7. 오늘 작성한 피보나치 수열의 재귀호출 방식의 동작을 분석해옵니다.  
   인자 6이 너무 빡세다 싶으면 5정도로 줄여서 진행해도 좋습니다.
   4는 안됩니다(볼 부분이 거의 없음)
8. 오늘 작성한 피보나치 수열의 재귀호출 방식을
   오늘 학습한 기계어 분석 관점에서 바라보도록 합니다.
```
