# 6 회차 숙제

```make
1. 오늘 학습한 벡터의 내적에 대해 조사 및 복습을 수행한다.
(어디에 사용되고 왜 사용하고 어떻게 응용할 수 있는지 등등)

2. 벡터에는 또 다른 곱셈 연산(외적)이 있다.
외적은 어떤 경우에 활용되고 어떻게 응용할 수 있는지 조사를 한다.
또한 우리가 작성한 벡터 예제에 벡터 외적 연산을 추가해보도록 한다(프로그래밍)

3. 작성한 행렬 코드의 곱셈 코드를 작성하세요.

4. 역행렬을 계산하는 코드를 작성하세요. (2 by 2)
   (3 by 3)은 determinant와 여인수 행렬을 사용해야하므로 패스합니다.

5. 오늘 학습한 구조체의 어셈블리 코드를 분석하세요.
   구조체가 어셈블리어로 표현하기 위해 어떻게 표현하는지를 파악합니다.

6. 오늘 마지막에 작성한 배열 포인터에 대한 개념을 잡기 위해
   마찬가지로 어셈블리 코드 분석을 수행해봅니다.
   (주석에도 적혀있듯이 포인터 배열과 배열 포인터는 다른 개념입니다)
```
