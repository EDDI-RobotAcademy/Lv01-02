리눅스 명령어

ls : 현재 디렉토리안의 파일 및 디렉토리를 보여주는 명령어이다.
 -a : 숨김파일을 포함하여 모든 파일 및 디렉토리를 보여준다.
 -R : 현재 위치부터 하위 디렉토리들의 파일을 보여 준다.
  
cd : 디렉토리를 이동한다.
ex)
cd ~/ : home 디렉토리로 이동
cd .. : 상위 디렉토리로 이동

mkdir : 디렉토리를 만드는 명령어이다.
 -p : 하위디렉토리를 만들며 실제 이름에 해당하는 디렉토리가 없으면 새로 생성해준다.

pwd : 디렉토리의 위치를 보여준다.

cp : 복사명령어이다.
 -r : 디렉토리 복사를 할때 사용하는 옵션이다.
